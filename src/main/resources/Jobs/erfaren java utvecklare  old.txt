Vi söker en erfaren Java utvecklare, enligt nedan: Beskrivning Uppdraget innebär att ingå ett utvecklingsteam på drygt 10 personer som arbetar med java-utveckling. Vi söker en ambitiös Java-utvecklare för att förstärka teamet som arbetar med utveckling av vårt system för bland annat automatiserade nätleveranser, nätinventariehantering, nätstatistik, integrationer med telefonisystem, samt vår kundweb. Vi strävar efter att ligga i teknikens framkant och arbetar i en Javabaserad Microservice-miljö där systemen huvudsakligen utvecklas med Java 8, Spring, Spring boot, jQuery och Angular, även tekniker såsom NodeJS och Ruby används. Utvecklingsmiljön är uppbyggd kring Intellij IDEA, Maven, Git och Gitlab.. Totalt arbetar ca 30 utvecklare och arkitekter på utvecklingsavdelningen.
