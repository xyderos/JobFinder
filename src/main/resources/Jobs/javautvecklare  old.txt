Till vår kund söker vi en Javautvecklare, enligt nedan: Uppdraget innebär att ingå ett utvecklingsteam på knappt 10 personer som arbetar med java-utveckling.  Vi söker en ambitiös Java-utvecklare för att förstärka teamet som arbetar med utveckling av kundens system för bland annat automatiserade nätleveranser, nätinventariehantering, nätstatistik, integrationer med telefonisystem, samt kunden kundweb. Vi strävar efter att ligga i teknikens framkant och arbetar i en Javabaserad Microservice-miljö där systemen huvudsakligen utvecklas med Java (vi börjar snart migrera till Java 10), Spring, Spring boot, jQuery och Angular, även tekniker såsom NodeJS och Ruby används.  Utvecklingsmiljön är uppbyggd kring Intellij IDEA, Maven, Git och Gitlab. Totalt arbetar ca 55 utvecklare och arkitekter på utvecklingsavdelningen. Dessa är uppdelade på mindre utvecklingsteam där utvecklingen sker enligt SCRUM.
